# Versu Documentation

Welcome to the Versu documentation site!

## What is Versu?

Versu is an intelligent versioning automation tool that eliminates manual version management. It analyzes your commit history using Conventional Commits, automatically determines semantic version bumps, manages dependency cascades, and generates changelogs.

## Key Features

✅ **Automatic Version Bumping** - Based on Conventional Commits
✅ **Multi-Module Support** - Each module versioned independently  
✅ **Dependency Cascade** - Automatic dependent versioning
✅ **Changelog Generation** - Beautiful changelogs from commits
✅ **Pre-release Support** - Alpha, beta, rc versions
✅ **Multiple Interfaces** - Library, CLI, GitHub Action
✅ **Extensible Architecture** - Custom adapters and plugins
