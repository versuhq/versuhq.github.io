# CLI Tool

Versu CLI provides a command-line interface for versioning your projects.

## Basic Usage

```bash
npx @versu/cli [options]
```

## Commands

### Default Command: Version

Run versioning for your project:

```bash
npx @versu/cli
```

This will:

- Analyze commits
- Calculate version bumps
- Update files
- Generate changelogs
